# Kevin Chromik Hackathon December 2025

*habe leider kaum Zeit gehabt, bitte erwartet kein UX / UI Feuerwerk*

## Beschreibung

Meistens schreibe ich meinen Brain Dump für die nächste Einkaufsliste über den Tag verteilt und das auch ziemlich chaotisch.

Diese App soll dabei helfen, die Einträge "effizient" zu gruppieren, damit man den Einkauf optimal erledigen kann. Beispielsweise werden alle Notizen für Gemüse in eine Gruppe gepackt, damit man nicht mehrfach zum Gemüsestand zurück muss.

Angebunden ist eine lokale AI über WebLLM, keine externen Aufrufe, aber :warning:  **ACHTUNG** :warning: , das Model ist sehr groß, besser am Handy nur mit WLAN ausprobieren.

## Technologien
- React
- Tailwind + HeroUI
- WebLLM
